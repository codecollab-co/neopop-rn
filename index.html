What's inside

Built to Ship

01

3D Surfaces

Real parallelogram rendering via @shopify/react-native-skia. Five-face depth model, crisp at any pixel density.

02

Fluid Animations

Reanimated 3 worklets on the UI thread. Zero JS bridge calls per frame, even at 120 FPS.

03

Complete Token System

Exported color palettes, spacing scale, typography — plus Style Dictionary output for CSS, Figma, Android, and iOS.

04

Gesture-Driven

Pan sliders, swipe rows, drag-to-dismiss sheets, velocity-aware carousels — all via RNGH v2.

05

Fully Typed

Strict TypeScript throughout. Every prop interface, every colorConfig key, every theme token exported.

06

WCAG 2.1 AA

accessibilityRole, accessibilityState, and accessibilityValue on every interactive component.
